Traffic Data Analysis System

Developed a Python-based traffic data analysis system that processes and visualizes vehicle movement data from two monitored junctions (Elm Avenue/Rabbit Road and Hanley Highway/Westway). The system reads CSV data files, performs comprehensive traffic pattern analysis, and generates statistical reports with histogram visualizations. The application validates user inputs, handles data processing for specific dates, and stores analyzed results for future reference.

Technical Implementation

  • Language: Python
  • Data Processing: CSV parsing and manipulation for multi-column traffic datasets
  • Validation Logic: Custom input validation functions with range checking and type verification
  • Data Structures: Efficient handling of traffic parameters including junction names, timestamps (24-hour format), vehicle classifications (truck, bicycle, motorcycle, scooter, bus), directional flow (in/out), speed metrics, and weather conditions
  • Visualization: Graphical histogram generation using Python plotting libraries
  • File Operations: Dynamic file naming with date formatting (DDMMYYYY), read/write operations, and error handling for missing datasets
